## 2. 范式
- **Zero-shot**
指在训练阶段不存在与测试阶段完全相同的类别，但是模型可以使用训练过的知识来推广到测试集中的新类别上。

- **Few-shotLearning**
指少样本学习，当模型在学习了一定类别的大量数据后，对于新的类别，只需要少量的样本就能快速学习，对应的有one-shot learning，单样本学习，也算样本少到为一的情况下的一种few-shot learning。

## 2. JSON(JavaScript Object Notation)
- 是一种轻量级的数据交换格式，易于人阅读和编写，同时也易于机器解析和生成。
- JSON使用**键值对**的形式来表示数据结构，支持多种数据类型，如字符串、数字、布尔值、数组和对象等。
- JSON广泛应用于Web开发中，用于在客户端和服务器之间传输数据, 结构化文本

### 1. json对象 < = > python字典   
关键区别在于：json对象的键必须是字符串，而python字典的键可以是任意不可变类型（如字符串、数字、元组等）。
在Python中，可以使用内置的**json模块**来处理JSON数据。以下是一些常用的操作：
```
# Python对象 → JSON字符串
json.dumps(
    字典或列表对象，
    ensure_ascii=False,  # 不要将非ASCII字符转义为\uXXXX格式，可以直接输出中文等非ASCII字符
)

# JSON字符串 → Python对象(字典或列表)。
    json.loads(JSON字符串)
```
